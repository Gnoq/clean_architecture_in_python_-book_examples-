============
Installation
============


Stable release
--------------

To install Clean Architecture using TDD in Python, run this command in your terminal:

.. code-block:: console

    $ pip install clean_arch_py

This is the preferred method to install Clean Architecture using TDD in Python, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Clean Architecture using TDD in Python can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Gnoq/clean_arch_py

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/Gnoq/clean_arch_py/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Gnoq/clean_arch_py
.. _tarball: https://github.com/Gnoq/clean_arch_py/tarball/master
